# MicroShell

A simple shell for MicroPython boards.

## Description

This is a basic shell project designed for embedded boards running MicroPython.

It is still under development and not complete yet.  
Currently, only some basic features are implemented.

## Requirements

- A board with MicroPython installed

## Usage

1. Copy `main.py` and `extra_.py` to your board
2. Run `main.py`

After that, you can use the shell.

## Notes

This project is still in an early stage.  
More features will be added in the future.
